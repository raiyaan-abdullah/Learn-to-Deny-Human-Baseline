# Human Baseline
[![Paper](https://img.shields.io/badge/arXiv-2606.31187-b31b1b.svg)](https://arxiv.org/abs/2606.31187)
This folder contains the browser-based multiple-choice quiz used to measure the human baseline reported in the paper. The quiz randomly presents five UCF101-AD action-denial examples and asks participants to select the caption that best describes each clip, including a “none of the choices” option when the expected action is not performed.
